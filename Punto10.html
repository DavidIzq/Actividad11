Ejemplos de Target

El valor "_blank" del atributo target permite el enlace sea mostrado en una nueva ventana del navegador.
Esto es un ejemplo, en el siguiente link Youtube se abrira en una nueva pestaña ABRIR LINK

El valor "_self" del atributo target el link será mostrado en la misma ventana donde está el enlace.
Esto es un ejemplo, en el siguiente link Youtube se abrira en la misma pestaña ABRIR LINK

El valor "_parent" del atributo target permite que el link sea mostrado en el FRAMESET padre del frame actual.
Esto es un ejemplo, en el siguiente link Youtube se abrira en la misma pestaña ABRIR LINK

El valor "_top" del atributo target permite que el enlace sea mostrado usando todo el espacio de la ventana del navegador destruyendo toda estructura deframes. Este valor debe ser usado siempre que creemos un enlace a una página externa a nuestro sitio web. Ejemplo: ABRIR LINK
